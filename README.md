# fifa-chn-guide

本仓库用于归档和发布多个独立 HTML 页面，内容涉及 FIFA 相关中文资料与指南。

## 项目简介

这是一个以 HTML 文件形式存放的静态资源仓库，主要用于收集、整理和分享与 FIFA 中文社区相关的信息页面。所有页面均为独立 HTML 文件，不依赖任何后端服务或特定域名。

## 目录说明

仓库目录结构如下：

```
/
├── README.md               # 本说明文件
├── pages/                  # 存放所有 HTML 页面
│   ├── index.html          # 页面索引
│   └── ...                 # 其他页面文件
└── assets/                 # 静态资源（图片、样式等）
```

- `pages/` 下存放所有归档的 HTML 页面，可通过本地浏览器直接打开。
- `assets/` 用于存放页面引用的图片、CSS 等资源文件。

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，不依赖外部在线资源（除非在页面内明确声明）。
- 页面内容可能包含中文翻译、资料整理、工具界面或信息汇总。
- 所有页面均不指向任何特定网站或域名，仅作为离线可用的参考资料。
- 页面可能不定期更新或替换，以保持内容准确性。

## 维护说明

- 本仓库由维护者根据社区反馈或自身整理需求更新。
- 欢迎通过 Issues 提交页面内容建议或错误反馈。
- 不接受任何形式的商业推广或域名植入。
- 若需引用或分发页面内容，请保留页面内原始注释与来源信息。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接使用浏览器打开 `pages/` 下的任意 HTML 文件。
3. 也可通过 GitHub Pages 等服务部署（需自行配置）。
